* 配置选择

** macports

使用 macports 替代原来的 homebrew ，用编译解决自动安装的问题，还可以放弃 ruby 相关的依赖，轻松。

** emacs

Emacs 的配置文件夹做为子项目保存，配置文件更新之后需要同步更新。

** python

使用 =uv= 工具管理 =Python= 相关的包和开发环境。

** flutter

安装 =fvm= 工具，用来管理 =flutter= 的包管理器

~fvm install stable~ 

** nodejs

安装 =votla= 工具，用来管理 =nodejs= 的相关工具，同时使用 =pnpm= 来替代 =npm= 。
