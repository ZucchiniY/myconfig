* 工具选择

** macports

使用 macports 替代原来的 homebrew ，用编译解决自动安装的问题，还可以放弃 ruby 相关的依赖，轻松。

** wezterm

终端模拟器，从 iTerm 和 Kitty 迁移而来。

** emacs

Emacs 的配置文件夹做为子项目保存，配置文件更新之后需要同步更新。

** python

使用 =uv= 工具管理 =Python= 相关的包和开发环境。

** flutter

安装 =fvm= 工具，用来管理 =flutter= 的包管理器

~fvm install stable~ 

** nodejs

安装 =votla= 工具，用来管理 =nodejs= 的相关工具，同时使用 =pnpm= 来替代 =npm= 。

** android

安装 android-sdk android-ndk 相关工具，使用 android-studio 最方便。

* 配置

** zsh

使用 starship 和 zsh 配置，删除 .tmux 和 ohmyzsh ，简化整体的配置。

- 新增 zoxide 命令

   #+begin_src shell
     # 安装 fzf 依赖
     sudo port install fzf # macos
     sudo dnf install fzf # fedora

     # 安装 zoxide 命令
     cargo install zoxide --locked
   #+end_src

   启动 zoxide ，向 =.zshrc= 中添加：

   #+begin_src shell
     # 启动 zoxide
     eval "$(zoxide init zsh)"

     # 加载 fzf 插件
     source /opt/local/share/fzf/shell/key-bindings.zsh
     source /opt/local/share/fzf/shell/completion.zsh
   #+end_src
